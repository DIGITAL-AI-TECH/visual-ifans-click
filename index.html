01 / Logo System

Variantes do Logo

Todas as variantes aprovadas para uso em diferentes contextos e fundos. Clique para baixar ou copiar a URL CDN.

Como usar cada variante

Primária

Use em fundos brancos e tons claros. É a versão padrão para materiais impressos, apresentações e interfaces digitais com fundo claro.

  • Documentos e apresentações
  • Fundos brancos e cinzas claros
  • Email marketing
  • Interface de produto (light mode)

Negativa / Branca

Use em fundos escuros, fotografias com fundo escuro e materiais onde a versão primária não teria contraste suficiente.

  • Fundos escuros e dark mode
  • Sobre fotografias escuras
  • Rodapé de sites
  • Brindes e merchandising escuro

Profundidade

Versão especial com elemento visual de profundidade. Use em contextos de destaque onde a marca deve ter impacto visual máximo.

  • Capas de apresentações
  • Materiais de alto impacto
  • Fundos com gradiente
  • Conteúdo premium e editorial

Símbolo / Ícone

Use quando o espaço é restrito ou a marca já é conhecida no contexto. Ideal para avatares, favicons e espaços quadrados.

  • Foto de perfil e avatares
  • Favicon do site
  • App icon
  • Espaços muito pequenos (<120px)

Guia de instalação

Gmail

  1. Abra Gmail → ⚙️ → Ver todas as configurações
  2. Aba Geral → role até Assinatura
  3. Clique em Criar nova
  4. No editor, clique em <> (código-fonte)
  5. Cole o HTML copiado da aba "Código HTML"
  6. Clique em Ok → Salvar alterações

Outlook

  1. Arquivo → Opções → Email → Assinaturas
  2. Clique em Novo e nomeie a assinatura
  3. No editor, clique em Editar no HTML
  4. Cole o código HTML da aba "Código HTML"
  5. Clique em OK e salve
  6. Selecione a assinatura padrão para novos emails

Apple Mail

  1. Salve o HTML como arquivo .html
  2. Abra o Terminal: open ~/Library/Mail/V10/MailData/Signatures/
  3. Copie o arquivo .html para a pasta
  4. Em Mail → Preferências → Assinaturas
  5. Crie nova assinatura e cole o HTML no campo

⚡ Personalizar a assinatura

No código HTML, substitua os seguintes campos:

  • SEU NOME COMPLETO → seu nome
  • Seu Cargo → seu cargo
  • +55 (64) 9XXXX-XXXX → seu telefone
  • seu@email.com.br → seu email
